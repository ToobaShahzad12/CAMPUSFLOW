# CampusFlow

CampusFlow is a modern student productivity dashboard designed to help students manage assignments, schedules, attendance, notes, and study progress in one place.

## Features

- Assignment Management
- Weekly Schedule Calendar
- Notes Section
- Study Metrics Dashboard
- Study Streak Tracking
- Attendance Heatmap
- Focus Session Card
- Responsive UI
- Modern Dashboard Design

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
- Git & GitHub
- Vercel
